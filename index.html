About Me

Olá! Sou Vitor, um estudante de engenharia de software de 18 anos. Estou entusiasmado com as possibilidades desta área e me dedico ao aprendizado constante. Meu primeiro contato com a programação foi antes da faculdade, o que despertou meu interesse e me motivou a seguir essa carreira. Atualmente, estou trabalhando com várias linguagens, incluindo HTML e CSS para criar interfaces web, JavaScript e Python para desenvolvimento tanto backend quanto frontend, e tenho também familiaridade com a linguagem C#. Estou sempre em busca de novos desafios e oportunidades para expandir meu conhecimento e aprimorar minhas habilidades.

Services

HTML

JavaScript

Como desenvolvedor, o JavaScript é uma das ferramentas mais poderosas que utilizo em meu trabalho. Enquanto o HTML cuida da estrutura e organização do conteúdo, o JavaScript adiciona interatividade e dinamismo às páginas da web.

Python

Python é uma linguagem versátil e poderosa que uso para desenvolvimento web, automação e análise de dados. Sua sintaxe clara e vasta gama de bibliotecas o tornam indispensável em meu trabalho diário.
